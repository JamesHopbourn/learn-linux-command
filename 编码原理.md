#### 结构概览
```
0000 0000 0000 0000 0000 0000 0000 0011
||                                    |
||--|---------------------------------|
|   |                 
|   +--- 后面 31 位为数据位，例如此处的数据为 (1*2^0)+(1*2^1)=3
|        计算机存储都是从0开始，最后一位是2的0次方，倒一位为2的1一次方
|        依次类推直到前面的第二位都是如此计算然后相加
|
+------- int 为有符号整型第一位数是符号位，0 位正数，1 为负数
         unsigned int 是无符号整型，32位全部都是数据位
```
