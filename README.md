# learn-linux-command
Linux 命令学习

安装全套 GNU 工具
```
brew install coreutils findutils gnu-sed gnu-indent gnu-tar gnu-which gnutls grep gzip screen watch wdiff wget less unzip
```

#### 文本处理大杀器
sed awk perl tr cat

#### 倒序 逆向 排序
rev tac sort

#### 文件查找字符查找
grep

#### 剪切版操作
pbcopy pbpaste
