# learn-linux-command  
  
#### 安装 GNU 工具  
```  
brew install coreutils findutils gnu-sed gnu-tar gnu-which grep gzip screen fswatch wdiff wget less unzip  
```

echo print printf tee

sed awk perl

tr  
wc  
pr  
seq  
cat rev  
cut paste split  
#### 行列相关  
tac  
unip sort  
head tail  
less more  
#### 文件查找字符查找  
find grep  
#### 剪切版操作  
pbcopy pbpaste
