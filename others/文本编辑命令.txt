文本编辑命令

------------------------------------------------------------
awk
awk -d ' ' '{print $1}'
------------------------------------------------------------
comm
basename
可用于 for 循环过滤出文件名而不用 sed
------------------------------------------------------------
sort
sort 文件名
sort -u 文件名 去除重复行
sort -u 文件名 -o 文件名
sort -n 文件名 正确排序数字
sort -k '' 文件名 指定排序所用行
sort -t '' 文件名 指定排序分隔符
------------------------------------------------------------
comm
  -1    不输出文件1 特有的行
  -2    不输出文件2 特有的行
  -3    不输出两个文件共有的行
------------------------------------------------------------
od
od [-A 地址进制] [-t 显示格式] 文件名
-A ( 地址进制)按指定的进制显示地址信息;
-t 指定数据显示的格式。

-A 指定的地址进制包括：
o：八进制（系统默认值）
d：十进制
x：十六进制
n：不打印位移值

-t 指定数据的显示格式的主要参数有：
c：ASCII字符或反斜杠序列(如\n)
d：有符号十进制数
f：浮点数
o：八进制（系统默认值）
u：无符号十进制数
x：十六进制数
------------------------------------------------------------
csplit
csplit 文件名 line-num
------------------------------------------------------------
cut
cut -d '' -f2-9 文件名 以空格为分隔符输出2-9列
------------------------------------------------------------
diff
diff 文件一 文件二
------------------------------------------------------------
fmt
指定字符长度重排文本：fmt -w num 文件名
------------------------------------------------------------
fold
限制文件列宽：fold -w num 文件名
------------------------------------------------------------
head
显示前面多少行：head -n num 文件名
显示除了多少行：head -n -num 文件名
显示前面多少字节：head -c num 文件名
显示除了多少字节：head -c -num 文件名
------------------------------------------------------------
iconv
转换编码：iconv 文件一 -f EUC-JP-MS -t UTF-8 -o 文件二 
------------------------------------------------------------
join
从两个文件中相同的地方开始合并，多用于行列数据
join 文件一 文件二 > 输出文件
------------------------------------------------------------
less
less 文件名
------------------------------------------------------------
join
------------------------------------------------------------
more
more 文件名
------------------------------------------------------------
nl
nl 文件名
------------------------------------------------------------
paste
paste 文件一 文件二 > 输出文件
------------------------------------------------------------
sed
sed -i '' -e 's/ / /' 文件名
------------------------------------------------------------
strings
查找二进制文件中所有可用于打印的字符
------------------------------------------------------------
tail
tail -F 文件名
------------------------------------------------------------
uniq
uniq 文件名 合并相邻重复行
sort 文件名|uniq 排序后合并重复行
uniq -c 文件名 统计重复行出现次数
uniq -d 文件名 显示所有重复的行
------------------------------------------------------------
wc
wc c:字节 l:行数 w:字数
------------------------------------------------------------
xargs
find . -name "*.m"|xargs basename
------------------------------------------------------------
tr
压缩字符：echo "thissss is      a text linnnnnnne." | tr -s ' sn'
替换字符：tr "\n" " " < source.txt > new.txt
删除字符：echo "hello 123 world 456" | tr -d '0-9'
